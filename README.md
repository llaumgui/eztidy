Introduction
============
A complete solution of tools for cleaning with Tidy the HTML code generated by eZ Publish 4.

eZTidy included
===============
 * Template operator : tidy()
 * Class for use in your code : eZTidy
 * Output filter (require eZ Publish 4.1.x)
